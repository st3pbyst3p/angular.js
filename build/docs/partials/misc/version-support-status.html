Version Support Status

AngularJS support has officially ended as of January 2022.

Visit angular.io for the actively supported Angular.

What does end of support mean?

The code will remain accessible on GitHub, npm, Bower, and Release archive. This website will remain here indefinitely.

The GitHub repository will be in an archived state, meaning that no new issues or pull requests can be submitted.

See https://goo.gle/angularjs-end-of-life for the full details.
